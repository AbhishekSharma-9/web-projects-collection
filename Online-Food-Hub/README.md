
# 🍔 Online Food Hub

Welcome to **Online Food Hub** — one of the best and most polished frontend web projects built by **Abhishek Sharma**. 🚀

This project showcases a **fully responsive** and **visually engaging** online food ordering interface that is optimized for **all screen sizes**, from mobile phones to large desktop monitors. It is a demonstration of clean HTML/CSS structuring, responsive media queries, and elegant design principles — making it a standout portfolio piece.

## ✨ Key Features

- 🔄 **Fully Responsive Design** – fluid layout with perfect scaling across screen sizes.
- 🍽️ **Attractive UI** – beautiful food cards, call-to-action buttons, and typography.
- 🎨 **Clean CSS Structure** – well-commented and modular stylesheet.
- 🚀 **Lightweight & Fast** – no heavy frameworks used; pure HTML and CSS.

## 🛠️ How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/AbhishekSharma-9/web-projects-collection.git
   ```
2. Navigate to the folder:
   ```bash
   cd web-projects-collection/Online-Food-Hub
   ```
3. Open `index.html` in any browser — and you're good to go!
